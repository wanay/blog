Javascript Variable

Nov 6th, 2014 4:05 am

ဒီနေ့တော့ javascript variable တွေအကြောင်းကိုလေ့လာကြပါမယ်။variable ဆိုတာက တန်ဖိုးတွေကို ယာယီသိုလှောင်ပေးထားတဲ့ ကိန်းရှင်တွေလို့ခေါ်ပါတယ်။javascript မှာထူးခြားချက် variable တွေရဲ့ data type ကိုကြိုတင်ကြေငြာပေးစရာမလိုပါဘူး။variable တစ်ခုထဲကို String တန်ဖိုးတွေ ထည့်ထားရင် သူ့ရဲ့ data type ကလည်း String data type ဖြစ်နေမှာပါ။တစ်ကယ်လို့ integer တန်ဖိုးတွေထည့်ထားခဲ့ရင်လည်း သူ့ရဲ့ data type က integer ဖြစ်နေမှာပါပဲ။ javascript မှာ variable တွေကို var ဆိုတဲ့ keyword နဲ့သတ်မှတ်ပါတယ်။ဥပမာအနေနဲ့ var i = “Hello javascript”; လို့ဆိုလိုက်တာနဲ့ i ဆိုတဲ့ variable ထဲကို Hello javascript ဆိုတဲ့ string တစ်ခုသိုလှောင်ပြီးသားဖြစ်သွားမှာပါ။တစ်ကယ်လို့ var x = 21; ဆိုတာနဲ့ x ဆိုတဲ့ variable ထဲကို 21 ဆိုတဲ့ integer တန်ဖိုးတစ်ခုသိုလှောင်လိုက်တာပါ။ဒါဆိုလက်တွေ့ code လေးတွေစမ်းရေးကြည့်ပါမယ်။

ပထမဆုံး variable x ရဲ့တန်ဖိုးကို 20 လို့သတ်မှတ်ပေးထားပါတယ်။နောက်တစ်ခါ variable y ရဲ့တန်ဖိုးကိုတော့ 28 လို့ပေးထားပါတယ်။နောက်ဆုံးမှာတော့ variable x နဲ့ variable y တို့ပေါင်းလို့ရတဲ့အဖြေကို variable z ထဲမှာသိုလှောင်လိုက်ပါတယ်။နောက်တော့ document.write(z); ဆိုပြီးတော့ variable z ထဲမှာသိုလှောင်ထားတဲ့တန်ဖိုးကို document.write နဲ့ output ထုတ်ပြလိုက်တာပါပဲ။အရမ်းလွယ်ပါတယ်။တစ်ခြား number တွေလည်းပြောင်းပြီးစိတ်ကြိုက် ပေါင်း နှုတ် မြောက် စား တွေလုပ်ကြည့်ကြပါ။ဒီနေရာမှာ ကျွန်တော်က x နဲ့ y ပေါင်းလို့ရလာမယ့်တန်ဖိုးကို variable z ထဲမှာကြိုတင်သိုလှောင်ထားလိုက်တာပါ။အဲဒါကြောင့် x နဲ့ y ပေါင်းလို့ရလာမယ့်တန်ဖိုးကို လိုချင်တိုင်း z ကိုခေါ်လိုက်တာနဲ့ လွယ်လွယ်ကူကူရလာမှာပါ။အဲလိုသာမရေးထားခဲ့ရင် x နဲ့ y ပေါင်းချင်တိုင်း x + y လိုက်ရေးနေရရင်အဆင်ပြေမှာမဟုတ်ပါဘူး။နမူနာအနေနဲ့ ဒီလိုဖြစ်နေမှာပါ။

variable တွေကို integer တွေပဲပေါင်း လို့ရတာမဟုတ်ပါဘူး။String type တွေကိုလည်း ပေါင်း လို့ရပါတယ်။ဥပမာအနေနဲ့

ဒါဆိုရင် မင်္ဂလာပါ javascript ဆိုတာကို output ထုတ်ပြပေးမှာဖြစ်ပါမယ်။မင်္ဂလာပါ နဲ့ javascript ကြားမှာ space လေးခြားချင်တာကြောင့် မင်္ဂလာပါ ဆိုတာရဲ့အနောက်မှာ space လေးခြားထားခဲ့တာသတိထား မိမယ်ထင်ပါတယ်။ဒါဆိုရင် int အချင်းချင်း string အချင်းချင်းပဲပေါင်းလို့ရတာလားဆိုတော့ မဟုတ်ပါဘူး။ int နဲ့string ကိုလည်းပေါင်းလို့ရပါတယ်။

နည်းနည်းလောက်ပြန်ရှင်းပြရရင် name ဆိုတဲ့ variable ထဲမှာ Wanay ဆိုတဲ့ string တစ်ခုကိုသိုလှောင်ထားလိုက်ပါတယ်။နောက်တစ်ခါ age ဆိုတဲ့ variable ထဲမှာတော့ 16 ဆိုတဲ့ int တစ်ခုကိုသိုလှောင်လိုက်ပါတယ်။အောက်ဆုံးမှာတော့ output ထုတ်ပြဖို့အတွက် document.write ထဲမှာ “Hi, I am “ ဆိုတဲ့စာသားနဲ့ name ဆိုတဲ့ variable ကို + ပေါင်းပါတယ်။နောက်တစ်ခါ “ and” ဆိုတဲ့စာသားနဲ့ ပေါင်းပါတယ်။ထပ်ပြီးတော့ age ဆိုတဲ့ variable ကို ထပ်ပေါင်းပါတယ်။နောက်ဆုံးမှာတော့ “ years old.” ဆိုတာနဲ့ပေါင်းလိုက်တဲ့အခါမှာတော့ output မှာ Hi, I am Wanay and 16 years old. ဆိုတာကိုတွေ့ရမှာပါ။ဒီနေ့တော့ variable အကြောင်း ဒီလောက်နဲ့နားပါအုံးမယ်။
