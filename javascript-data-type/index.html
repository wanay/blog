Javascript Data Type

Sep 15th, 2014 4:12 am

ဒီနေ့တော့ javascript variable အကြောင်းတွေ စလေ့လာကြည့်ကြရအောင်။ Variable တွေကို Global variable နဲ့ Local variable ဆိုပြီးခွဲခြားထားပါတယ်။Global variable ဆိုတာကတော့ function တွေရဲ့အပြင်ဘက်မှာ ကြေငြာထားတဲ့ variable တွေကိုခေါ်ပါတယ်။သူ့ကိုလက်ရှိ Document ထဲကနေပြန်ပြီးစိတ်ကြိုက်ခေါ်သုံးလို့ရပါတယ်။Local variable ဆိုတာကတော့ function တစ်ခုထဲမှာ ကြေငြာထားတဲ့ variable တွေကိုခေါ်ပါတယ်။သူ့ကိုတော့ သက်ဆိုင်ရာ function အောက်မှာ ပြန်ခေါ်သုံးလို့ရပါတယ်။အဲဒီ function အပြင်ကနေလှမ်းခေါ်သုံးလို့မရပါဘူး။အခုဆက် ပြီးတော့ Data type တွေအကြောင်းဆက်လေ့လာကြည့်ပါမယ်။Data type သုံးမျိုးရှိပါတယ်။ Primary Data Type Composite Data Type Special Data Type ဆိုပြီး(၃) မျိုးသတ်မှတ်ထားပါတယ်။ Primary Data Type ထဲမှာမှ Numbers String Boolean ဆိုပြီး သုံးမျိုးထပ်ခွဲထားပါတယ်။Numbers Data Type ဆိုတာကတော့ Number တွေပေါ့ဗျာ :P ။Number ထဲမှာလည်း ကိန်းပြည့်တွေဆိုရင် Integer လို့ခေါ်ပါတယ်။ Decimal တွေဆိုရင်တော့ Float လို့ခေါ်ပါတယ်။နောက်ပြီးတော့ String ဆိုတာကတော့ စာသားတန်ဖိုးတွေကို ခေါ်ပါတယ်။Boolean Data type မှာတော့ true/false ဆိုတဲ့ တန်ဖိုးနှစ်ခုပဲရှိပါတယ်။ Composite Data Type Composite Data Type မှာတော့ Object နဲ့ Array တန်ဖိုးတွေပါ ပါတယ်။အဲဒီအပိုင်းကိုတော့ နောက်ပိုင်းမှာ သေသေချာချာ ထပ်ရှင်းပါမယ်။ Special Data Type Special Data Type မှာတော့ null နဲ့ undefined ဆိုတာတွေပါ ပါတယ်။null ဆိုတာကိုတော့ တစ်ချို့က ဘာတန်ဖိုးတွေမှ မပါတဲ့ variable လို့လည်းပြောကြတယ်။နောက်တစ်ချို့တွေကလည်း number ဆိုရင် default တန်ဖိုးက (0) လို့လည်းပြောကြတယ်။Boolean မှာတော့ default အနေနဲ့ false လို့သတ်မှတ်တာတွေလည်းရှိပါတယ်။

browser မှာ right click နှိပ်ပြီး inspect element က console မှာသွားကြည့်ရင် ၀ ဆိုတာကိုတွေ့ရပါလိမ့်မယ်။

အပေါ်က program run ပြီး inspect element က console မှာသွားကြည့်လိုက်ရင်တော့ 3 ဆိုတာကိုတွေ့ရမှာပါ။ေနာက္တစ္ခါ Undefined ဆိုတာကတော့ သူ့ကိုဘာတန်ဖိုးမှ မသတ်မှတ်ရသေးတဲ့ variable ကိုဆိုလိုပါတယ်။

ဒီ program ကို run ပြီး inspect element မှာသွားကြည့်ရင်တော့ Type of x is undefined ဆိုတာကိုမြင်ရမှာပါ။ ဒီနေ့တော့ဒီလောက်နဲ့ပဲနားကြရအောင်ဗျာ။နောက်နေ့တွေမှာတော့ အခု Data Type တွေကို code တွေနဲ့သေချာစမ်းပြီး လေ့လာကြပါမယ်။
