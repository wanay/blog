Angular Fundamental 3

Nov 8th, 2014 4:21 pm

Angular အကြောင်းမရေးဖြစ်တာတော်တော်ကြာသွားကြောင့် အရင်အပိုင်းတွေမှာရေးပြခဲ့တာတွေက angular 1.3 နောက်ပိုင်းမှာ အလုပ်မလုပ်ဘူးဖြစ်နေတဲ့အကြောင်းလေးလည်းရေးချင်တာနဲ့ ပြန်ရေးလိုက်ပါအုံးမယ်။အခုနောက်ပိုင်း angular 2.0 ဆိုတာကလည်း လူပြောများနေကြတဲ့ ကိစ္စတစ်ခုပါပဲ။ဘာလို့လဲဆိုတော့ angular 1.3 နောက်ပိုင်းကနေ အရင် တုန်းက version တွေနဲ့နောက်ကွယ်မှာတော်တော်လေးကိုပြောင်းလဲသွားပါပြီ။ဝမ်းသာစရာ သတင်းလို့ပဲ ဆိုရမလားမသိဘူး IE 8 ကို angular 1.3 စပြီး support မလုပ်တော့ပါဘူး။ဒီမှာတော့ အရင်အပိုင်းက code တွေဘာလို့အလုပ်မလုပ်ရတာလဲ ဆိုတာပဲအဓိကပြောသွားပါမယ်။အရင်တုန်းကအပိုင်းတွေမှာ ng-contorller တွေကို global ကြေငြာထားခဲ့ပါတယ်။အဲဒါကြောင့်အခုနောက်ပိုင်း angular 1.3 မှာအလုပ်မလုပ်တော့တာပါ။အရင်အပိုင်းက နမူနာပေးခဲ့တဲ့ code တွေကိုနည်းနည်းလောက် ထပ်ဖြည့်ပြီးရှင်းပြပါမယ်။

အများကြီးထူးထူးခြားခြားမပြင်လိုက်ပါဘူး။အရင်က ရှိခဲ့တဲ့ search မှာ model တွေကိုရှာမယ်။နောက်မရှိသေးဘူးဆိုရင် ဘေးက add button ကိုနှိပ်ပြီးအသစ်ထပ်ထည့်မယ်။နောက် input မှာဘာမှ မထည့်ထားဘဲ add ကိုနှိပ်လို့မရအောင် disable လုပ်ထားပါတယ်။အပေါ်ဆုံးမှာတော့ list ထဲမှာ model တွေဘယ်လောက်ရှိသလဲ count လုပ်ထားပါတယ်။ပထမဆုံး script ထဲက code တွေအကြောင်းစပြီး ပြောပါမယ်။

အရင်ဆုံးအပေါ်ဆုံးမှာ ဆိုပြီး myapp ဆိုတဲ့ module ကိုသုံးထားတာသတိပြုပါ။ နောက် script ထဲမှာ angular.module(‘myapp’, []) ဆိုတာကတော့ ‘myapp’ ဆိုတဲ့ angular module တစ်ခု ဖန်တီးလိုက်ပါတယ်။နောက် [] ထဲမှာ ဘာမှ inject မလုပ်ထားပါဘူး။နောက် controller(‘Ctrl’, function ($scope) ဆိုတာကတော့ ‘Ctrl’ ဆိုတဲ့ controller တစ်ခုဖန်တီးလိုက်တာပါ။ကျန်တာတွေကတော့ ပြီးခဲ့တဲ့အပိုင်း မှာပြောထားခဲ့ပြီးသားပါ။နောက်ပြီးတော့ နောက်ထပ် model တွေကိုထပ်ထည့်ဖို့အတွက် add ဆိုတဲ့ function တစ်ခုဖန်တီးထားပါသေးတယ်။

$scope.models.push($scope.search); ဆိုတာကတော့ search ထဲမှာရိုက်ထားတာတွေကို models array ထဲကို javascript ရဲ့ push method သုံးပြီးထည့်ပေးလိုက်တာပါ။ $scope.search = “”; ဆိုတဲ့ string အလွတ်တစ်ခုထည့်ထားတာကတော့ search ထဲမှာကျန်ခဲ့တဲ့ string တွေကိုရှင်းပစ်ဖို့အတွက်သုံးထားတာပါ။ အပေါ်ဘက်က html တွေကိုပြန်ကြည့်ရင် list ထဲက model တွေကို count လုပ်ဖို့အတွက် javascript ရဲ့ length ဆိုတဲ့ method ကိုသုံးပြီး data bind လုပ်ထားပါတယ်။

နောက်တစ်ခါ form ရဲ့ name ကို myform လို့ပေးထားပြီးတော့ ng-submit မှာတော့ script ထဲမှာရေးထားခဲ့တဲ့ add() ဆိုတဲ့ function ကိုလှမ်းခေါ်ထားပါတယ်။နောက်ပြီးတော့ input ထဲမှာဘာမှမထည့်ပဲ add လုပ်လို့မရအောင် required ဆိုတာထည့်ထားပါတယ်။button ထဲမှာတော့ ng-disabled=“myform.$invalid” လို့ထည့်ထားတဲ့အတွက် input ထဲမှာဘာတန်ဖိုးမှမထည့်ထားရင် add ဆိုတဲ့ button ကမှိန်နေပါလိမ့်မယ်။

ဒါဆိုရင်တော့ရေးထားခဲ့တဲ့ code တွေအကြောင်းခြုံငုံမိမယ်ထင်ပါတယ်။Controller တွေကို global ကြေငြာလို့မရတော့ဘူးဆိုပေမယ့် module တွေကိုတော့ရပါသေးတယ်။အပေါ်မှာရေးပြခဲ့တဲ့ script ကိုပဲ အခုလိုမျိုးလည်းရေးလို့ရပါတယ်။အလုပ်လုပ်ပုံခြင်းကတော့အတူတူပါပဲ။
