Hello Ruby 2

Nov 5th, 2014 10:22 pm

အခုကျွန်တော်တို့ နောက်ထပ်အနေနဲ့ ပေါင်း နှုတ် မြှောက် စား တွေလုပ်ကြည့်ရအောင်ပါ။ irb ထဲမှာ

လို့ရိုက်လိုက်ရင် 324 ဆိုတာကို output ထုတ်ပေးပါလိမ့်မယ်။ဒီနေရာမှာလည်း Number တွေကို Intege data type လို့သတ်မှတ်ပါတယ်။နောက်ထပ်

ဆိုရင်တော့ 222 ကို output ထုတ်ပေးပါလိမ့်မယ်။

ဒါဆိုရင်တော့ 312000 လို့ output ထွက်လာပါလိမ့်မယ်။သိပ်ထူးထူးခြားခြားမရှိလို့ နားလည်မယ်ထင်ပါတယ်။

ဒီနေရာမှာ ထူးခြားတာက 20 ကို 3 နဲ့စားတဲ့နေရာမှာ 6 လို့ပဲအဖြေထွက်လာပါတယ်။ဘာကြောင့်လဲဆိုတော့ ကျွန်တော်တို့က integer type ကိုသုံးထားတာကြောင့် integer type ကို output ပြန်ထုတ်ပေးတာဖြစ်ပါတယ်။ တစ်ကယ်လို့ကျွန်တော်တို့က Decimal တန်ဖိုးနဲ့လိုချင်ရင်တော့

ဒါဆိုရင်တော့ ကျွန်တော်တို့လိုချင်တဲ့ Decimal တန်ဖိုးနဲ့ output ထွက်လာပါပြီ။အဲဒီ Decimal တန်ဖိုးတွေကို Float data type လို့ခေါ်ပါတယ်။နောက်ထပ် Data type တစ်ခုဖြစ်တဲ့ Boolean ဆိုတာကတော့ true/false တန်ဖိုးတွေကိုသတ်မှတ်ပါတယ်။

ဒီနေရာမှာ 3 က 4 ထက်ငယ်တဲ့အတွက် Boolean type ဖြစ်တဲ့ false ကို output ထုတ်ပေးတာပါ။

ဒီမှာတော့ 4 က 3 ထက်ကြီးတဲ့အတွက် true ကို output ထုတ်ပေးပါတယ်။ ဒါဆိုရင်တော့စိတ်ကြိုက်တစ်ခြား ပေါင်း နှုတ် မြှောက် စား တာတွေလုပ်ကြည့်လိုက်ပါ။နောက်ထပ်အနေနဲ့ Modulus ဆိုတဲ့ % ကိုတော့ အကြွင်းရှာတဲ့နေရာမှာသုံးပါတယ်။

75 ကို 2 နဲ့စားရင် အကြွင်း 1 ရတာကြောင့် 1 ကို output ထုတ်ပေးပါလိမ့်မယ်။နောက်တစ်ခုအနေနဲ့ Number တစ်ခုရဲ့ power ကိုရှာချင်တဲ့အခါမှာ ** ဆိုတာကိုသုံးပါတယ်။

2 power 3 အဖြေကိုရှာလိုက်တဲ့အခါမှာ 8 ကို output ထုတ်ပေးပါတယ်။ကျွန်တော်တို့အပေါ်မှာ သုံးခဲ့တဲ့ +, -, *, /, %, ** ဆိုတဲ့ Operator တွေကို Arithmetic operator တစ်နည်းအားဖြင့် Math operator တွေလို့ခေါ်ပါတယ်။ဒီနေရာမှာ variable ကိုသုံးပြီးတစ်ခြား assignment operator တွေကိုရှင်းပြချင်တာကြောင့် variable ဆိုတာကို တန်ဖိုးတွေကိုယာယီသိုလှောင်ထားတဲ့ ကိန်းရှင် တစ်ခုလို့လွယ်လွယ်မှတ်ထားစေချင်ပါတယ်။အခုနောက်ထပ် addition assignment ဆိုတဲ့ += အကြောင်းကိုပြောပါမယ်။

ပထမဆုံး i ဆိုတဲ့ variable တန်ဖိုးကို 5 လို့သတ်မှတ်လိုက်ပါတယ်။အဲဒီ i ဆိုတဲ့ variable ထဲကိုပဲ i += 20 ဆိုပြီး 20 ထပ်ပေါင်းထည့်လိုက်ပါတယ်။အဲဒါကြောင့်နောက်ဆုံးမှာ variable ဖြစ်တဲ့ i ကိုခေါ်လိုက်တာနဲ့ သူ့ရဲ့ တန်ဖိုးက 25 ဖြစ်သွားပါတယ်။အရမ်းလည်းမခက်ခဲတာကြောင့် တစ်ခြား -= တို့ *= နဲ့ /= အကြောင်းတို့ကိုဆက်မရှင်းပြတော့ပါဘူး။ကိုယ်တိုင်သာစမ်းကြည့်ကြလိုက်ကြပါ။
