Angularjs Fundamental 2

Sep 20th, 2014 4:15 am

အခုဒုတိယအပိုင်းအနေနဲ့ Data binding နဲ့ပတ်သက်တာလေးနည်းနည်းထပ်ပြောပါအုံးမယ်။အရင်အပိုင်းကရှိထားပြီးသား Code ထဲကို အခုနောက်ထပ် Age ဆိုတဲ့ input တစ်ခုထပ်ထည့်လိုက်ပါမယ်။

Browser မှာသွားစမ်းကြည့်တဲ့အခါမှာ age ရဲ့ input မှာထည့်သမျှကိုကျွန်တော်တို့ age ကို * 2 ဆိုပြီးရေးထားတာကြောင့် အောက်က h2 tag ထဲမှာနှစ်ဆဖြစ်နေတာကိုတွေ့ရမှာပါ။ပြောချင်တာကတော့ Data bindingလုပ်တဲ့နေရာမှာ ဒီလိုမျိုးတွေလည်း လုပ်လို့ရတယ်ဆိုတာကိုသိစေချင်တာပါ။ ကျွန်တော်တို့နောက်ထပ်အသစ်တစ်ခုထပ်လေ့လာကြည့်ပါမယ်။

သိပ်ပြီးထူးထူးခြားခြားမဟုတ်ပါဘူး။$watch ကို input ထဲမှာ user ကရိုက်သမျှကို Console ထဲမှာစောင့်ကြည့်ဖို့အတွက်သုံးလိုက်တာပါ။Browser ရဲ့ inspect element ရဲ့ console မှာသွားကြည့်ရင် input ထဲမှာရိုက်သမျှကို watch လုပ်ပေးထားတာကိုတွေ့ရမှာပါ။အခုနောက်ထပ်Angular array အကြောင်းကိုဆက်ပြီးလေ့လာကြည့်ပါအုံးမယ်။

ကျွန်တော်တို့Model တွေရဲ့ နာမည်တွေကို List နဲ့ပြပေးဖို့ ul နဲ့ li ကိုသုံးလိုက်ပါတယ်။နောက်ပြီးli ထဲမှာ ng-repeat မှာ model in models လို့ရေးထားပါတယ်။ပထမဆုံး model ဆိုတာက ကျွန်တော်တို့Data bind မှာပေးထားတဲ့ ကိုရည်ညွှန်းပါတယ်။ကိုယ်ကြိုက်တဲ့နာမည်တွေပေးထားလို့ပါတယ်။နောက်ထပ်models ဆိုတာကတော့ အောက်က Script မှာရေးထားတဲ့ array list က models ကိုရည်ညွှန်းပါတယ်။တစ်ခြားနာမည်တွေလညး်ပေးလို့ရပါတယ်။ကျွန်တော်တို့Browser မှာသွားစမ်းမယ်ဆိုရင် $scope.models မှာပေးထားခဲ့တဲ့ နာမည်တွေကို Loop လုပ်ပြီးပြပေးပါလိမ့်မယ်။Javascript လေ့လာဖူးသူတိုင်း Array တွေအကြောင်းနာလည်ပြီးသားဖြစ်မယ်လို့ထင်လို့ ဒီနေရာမှာမပြောတော့ပါဘူး။အခုနောက်ထပ်ကျွန်တော်တို့Filter အကြောင်းဆက်ရေးပါမယ်။

အပေါ်ကprogram မှာ input တစ်ခုထပ်ထည့်ထားပါတယ်။နောက် search ဆိုတဲ့ ng-model တစ်ခုထည့်ပါတယ်။နောက်ပြီးတော့input ထဲမှာရှာသမျှ ပြပေးဖို့အတွက် filter:search ဆိုတာကိုသုံးလိုက်တာနဲ့User ကရိုက်ပြီး ရှာသမျှ တူတာတွေကို ပြပေးပါလိမ့်မယ်။အခုတော့ ဒီလောက်နဲ့ပဲနားပါအုံးမယ်။
